# LearningLua 
![Lua](https://img.shields.io/badge/Lua-2C2D72?style=for-the-badge&logo=lua&logoColor=white)

A simple Lua repository containing basic examples like Hello World, created for learning purposes.


### I'm using ![Lua](https://img.shields.io/badge/Lua-5.4-blue?logo=lua)

To install Lua in Linux:

```bash
sudo apt install lua5.4
```
To run a file:

```bash
lua filename.lua
```
